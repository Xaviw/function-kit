# 工具库文档

[Vitepress](https://vitepress.dev/zh/)

## 项目结构

```sh
├── .vitepress    # vitepress 配置
├── typedoc       # 自动生成的函数库接口文档
├── poster        # Canvas 海报文档
├── public        # 同 vue 项目 public
├── index.md      # 文档首页
└── tsconfig.json # TS 配置
```

## 常用命令

```sh
# 运行文档
pnpm dev

# 构建文档
pnpm build

# 预览构建产物
pnpm preview
```
